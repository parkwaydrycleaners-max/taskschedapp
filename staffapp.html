Firebase Sync Settings

Connect to Firebase Realtime Database for automatic syncing across all devices.

How to set up Firebase
  1. Go to console.firebase.google.com and create a new project
  2. In the sidebar, click Build > Realtime Database
  3. Click Create Database and choose a location
  4. Set the security rules to allow public read/write:
    • { "rules": { ".read": true, ".write": true } }
  5. Copy the Database URL (e.g. https://myproject-default-rtdb.firebaseio.com)
  6. (Optional) For private databases, use a Database Secret from Project Settings > Service Accounts
