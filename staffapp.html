How to set up Airtable
  1. Go to airtable.com and create a new Base
  2. Create a table named Checklist
  3. Add these fields:
    • RecordID (Single line text) - Primary field
    • Data (Long text)
    • LastModified (Number)
  4. Go to airtable.com/create/tokens
  5. Create a Personal Access Token with these scopes:
    • data.records:read
    • data.records:write
  6. Add your base to the token's access list
  7. Copy the Base ID from the URL (starts with "app")

Quick Reconnect: Save your config file once, then import it anytime to reconnect instantly.

Connection may reset on page refresh due to browser security. Keep your config file handy!
